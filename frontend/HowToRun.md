# How to Run

## Backend Setup

```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload
```

## Frontend Setup
new terminal

```bash
cd frontend
npm install
npm run dev
```

`cd frontend`
`npm install`
`npm run dev`
